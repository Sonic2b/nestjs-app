# 00-first-step

## Description

nestjs app demo 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Format codes

```bash
# format code
$ npm run format

# tslint rules check
$ npm run tslint-check
```
